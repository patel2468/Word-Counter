File Installation Key

A File Installation Key identifies the products you can install. You must use a File Installation Key if you do not have an Internet connection. The MATLAB® administrator can retrieve the File Installation Key from the License Center at the MathWorks® website.

Using a File Installation Key requires you to run the installer from either:

  • MATLAB DVD 1.

  • A folder that includes the installer and the product files.

    The MATLAB administrator can run the installer to download the required files to this folder.
